# MANUAL DO USUÁRIO

Bem-vindo(a) ao **Sistema de Gestão Financeira da Igreja**. Este manual foi feito para orientá-lo(a) nas tarefas diárias.

## 1. Como Acessar e Fazer Login
Abra o navegador e acesse o endereço fornecido pela equipe de TI. 
Clique em "Sign in" para se identificar. O acesso pode ser restrito conforme o seu nível de permissão:
- **Visualizador:** Apenas lê dados (não pode alterar).
- **Tesoureiro:** Registra entradas, saídas e emite relatórios.
- **Admin:** Controle total do sistema, incluindo acessos de usuários.

## 2. Como Usar as Funcionalidades

### Gestão de Entradas
1. Clique em **Entradas** no menu lateral.
2. Para adicionar, clique no botão superior direito **Nova Entrada**.
3. Preencha os campos obrigatórios (Data, Categoria, Valor e Forma de Pagamento) e confirme.
4. Você pode associar a entrada a um *Membro* se desejado.

### Gestão de Despesas
O processo é igual ao de Entradas, mudando o foco para as categorias (água, luz, aluguel, salários, etc). As saídas aparecem descontando os gráficos do Dashboard.

### Centros de Custo
Gerencie departamentos (ex: Jovens, Mulheres, Crianças, Missões). Você pode definir "Tetos de Gastos" (Orçamentos) para cada centro.

### Relatórios Gráficos e em PDF
Vá até a aba **Relatórios**. O sistema permite visualizar o cruzamento de receitas x despesas através de datas. É possível baixar a tela do relatório clicando em baixar (PDF).

## 3. Resolução de Dúvidas
Se o sistema apresentar tela branca e aviso vermelho, ele estará tentando te proteger de algum conflito (como salvar letra em campo de número). Preste atenção nas mensagens pequenas que surgem no canto superior (Toasts).
Em caso de esquecimento de senhas ou troca de papéis, fale com o Administrador atual para ser promovido de Visualizador a Tesoureiro.
